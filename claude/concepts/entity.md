# Entity（实体）

## 定义

`Entity` 是系统中**一切可拥有 DID 身份的对象**的统一抽象，并不特指人类用户。

## 包含范围

- **普通用户**：注册了 DID 账号的人
- **AI 对象**：由商户或平台创建的 AI 智能体，在系统中与普通用户平等对待
- **IoT 托管对象**：硬件设备、机器人等受托管的实体
- **未来可能的其他类型**

## 数据结构

`Entity` 是一个简洁的摘要结构，用于在列表、引用、消息发送方/接收方等场景中标识一个实体：

```protobuf
message Entity {
  EntityType type = 1;  // 实体类型（用户/AI/IoT等）
  string did  = 2;      // DID 身份标识符
  string name = 3;      // 显示名称
  string avatar = 4;    // 头像
  int64 update_time = 5;
}
```

## 命名说明

- **不使用 `User`**：user 在英文中偏指人类用户，语义过窄
- **不使用 `Unit`**（旧名）：unit 意为"单元"，语义模糊
- **使用 `Entity`**：身份/数据系统中的标准术语，覆盖任意可识别对象
